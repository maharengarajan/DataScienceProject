# DataScienceProject

## install uv package
```
pip install uv
```

## create virtual environment
```
uv venv
```

## activate virtual environment
```
.venv\Scripts\activate
```

## inialize uv project
```
uv init
```

## install requirements
```
uv add -r requirements.txt
```

## run the project
```
python -m src.datascience.utils.common or uv run -m src.datascience.utils.common
```
